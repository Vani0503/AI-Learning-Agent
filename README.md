# AI Learning Agent (n8n)

## Goal
Build a personal AI agent that filters, summarizes, and helps me learn what's happening in AI.

## Problem
- Too much AI content
- Hard to identify what matters
- Passive consumption → low retention

## MVP Scope
- Input: newsletters (Gmail)
- Process: filter + summarize
- Output: daily digest (Slack/Telegram)

## Future Scope
- quizzes (active recall)
- learning tracker
- weekly reflection

## Tech Stack
- n8n
- OpenAI / Claude
- Gmail
- Slack / Telegram

Not a summarizer → a learning system
